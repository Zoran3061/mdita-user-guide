# MDITA User Guide

A responsive web-based user guide designed to help professors and users understand and use the MDITA application.
The guide provides clear navigation, structured instructions, and visual examples through a simple web interface.

This project was created as part of an academic assignment and represents a documentation-style web application
built with pure frontend technologies.

## Features
- Structured user guide with multiple sections
- Sidebar navigation with nested topics
- Dynamic content switching without page reload
- Image preview and zoom functionality
- Responsive layout for desktop and mobile devices
- Focus on usability and clarity

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript (embedded in HTML)

## Project Structure
-index.html
-style.css
-script.js
-Slike/
-images used inside the guide
-Screenshots/
-application preview images
-README.md

## Run Locally
No build tools or dependencies are required.

Simply open the `index.html` file in your web browser.
